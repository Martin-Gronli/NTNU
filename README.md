# NTNU

Inneholder fag og prosjekter fra bachelorstudiet på NTNU.

## Mapper

### [BetterYoutubeVolumeScroll](BetterYoutubeVolumeScroll/)
Et userscript (Tamper, Violent- eller Greasemonkey) som lar deg justere volumet på YouTube ved å scrolle over videospilleren.

- **Fil:** `BetterYoutubeVolumeScroll.js`
- Skroller opp/ned for å øke/senke volumet med en konfigurerbar mengde (standard: 2%)
